这是我的一个自建项目。

我使用网心云 OEC 刷入 Armbian，在本地完成 Hugo 博客的初始化、主题迁移与内容管理，
再通过 GitHub 与 Cloudflare Pages 完成自动部署，并接入自定义域名。

这个项目的重点不只是“搭起来”，而是尽可能做到：

- 低成本
- 低功耗
- 维护简单
- 可持续更新
- 具备个人表达空间
