# 开始搭自己的博客

这个博客终于正式上线了。

它运行在 Hugo 生成的静态站之上，通过 GitHub 管理内容，并发布到 Cloudflare Pages。
对我来说，这不只是一个网站，更像是一个长期可维护的数字空间。

## 为什么要自己搭博客

有几个原因很直接：

1. 内容更可控  
2. 不依赖单一平台  
3. 可以按自己的审美和需求定制  
4. 折腾过程本身就很有意思

## 这次部署的核心流程

这次最终跑通的路线是：

- 本地设备写内容
- Hugo 生成静态页面
- Git 提交到 GitHub
- Cloudflare Pages 自动部署
- 自定义域名接入

## 接下来会做什么

接下来我会继续完善这些部分：

- 首页视觉
- 关于页
- 评论系统
- 更统一的配色
- 更多折腾记录和文章内容

## 写在最后

先把站搭起来，再慢慢把它变成自己喜欢的样子。
